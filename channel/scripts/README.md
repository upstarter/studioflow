# Channel Scripts Directory

## Overview

This directory contains production-ready scripts for channel episodes. Each script serves **dual purpose**:
1. **Real Channel Content** - Actual episodes for your YouTube channel
2. **Test Fixtures** - Comprehensive testing of StudioFlow system

## Script Structure

All scripts follow this format:
- **EP###_descriptive-title.md**

Example: `EP001_i-built-ai-that-edits-videos.md`

## Script Components

Each script includes:
1. **Production Details** - Episode number, title, duration, purpose
2. **YouTube Optimization** - Titles, thumbnails, descriptions, tags
3. **Full Production Script** - Complete script with markers
4. **Production Notes** - Recording setup, B-roll, editing notes
5. **Testing Focus** - Edge cases and system testing

## Using These Scripts

### For Recording

1. **Choose Script**: Select episode based on sequence
2. **Review Script**: Read full script, understand flow
3. **Prepare Assets**: Gather B-roll, screen recordings, graphics
4. **Record**: Follow script exactly, use markers as specified
5. **Test**: Use as test fixture for StudioFlow
6. **Edit**: Polish and publish as real episode

### For Testing

1. **Record with Markers**: Use exact marker commands
2. **Test System**: Run through StudioFlow pipeline
3. **Verify Results**: Check transcription, segments, rough cuts
4. **Identify Issues**: Note any bugs or edge cases
5. **Iterate**: Fix issues, improve system
